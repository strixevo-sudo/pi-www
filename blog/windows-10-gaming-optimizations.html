Windows 10 Gaming Optimizations That Actually Matter

25 Mar 20265 min readby STRIXEVO

Most Windows 10 optimization guides are full of placebo tweaks and registry voodoo that do nothing. This is not that. These are the changes I ran on my own machine - Ryzen 5 5600X, RTX 3050 8GB, 32GB RAM - and every single one has a real reason behind it.

Step 1 - Slash Your Startup Programs

Windows boots and silently launches a load of programs you never asked for. Every one of them is sitting in memory, eating CPU cycles, and competing with your game.

Open PowerShell and run this to see exactly what's auto-starting:

Get-CimInstance Win32_StartupCommand | Select-Object Name, Command, Location | Format-Table -AutoSize

Go through the list honestly. Ask yourself - do I need this running before I even open it Steam, Discord, your peripheral software - maybe. But NVIDIA Broadcast, your torrent client, extra game launchers, RGB software Open them when you need them.

To disable something from startup via PowerShell:

Remove-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Run" -Name "AppNameHere"
// Note System entries like SecurityHealth and your audio driver (RtkAudUService) live in HKLM and need admin PowerShell. Don't touch those - they're not the problem.

Step 2 - Enable Game Mode

Game Mode tells Windows to prioritize your game's threads over background processes. It's off by default on some machines. Turn it on:

Set-ItemProperty -Path "HKCU:\Software\Microsoft\GameBar" -Name "AllowAutoGameMode" -Value 1 Set-ItemProperty -Path "HKCU:\Software\Microsoft\GameBar" -Name "AutoGameModeEnabled" -Value 1

Step 3 - Enable HAGS

Hardware Accelerated GPU Scheduling moves GPU memory management off the CPU and onto the GPU itself. On modern hardware - RTX 2000 series and up, RX 5000 series and up - this reduces CPU overhead and can lower latency.

Run this in an admin PowerShell:

Set-ItemProperty -Path "HKLM:\SYSTEM\CurrentControlSet\Control\GraphicsDrivers" -Name "HwSchMode" -Value 2

Reboot after. It won't take effect until you do.

// Note If you're on older hardware or running an older driver, HAGS can cause instability. If you see issues after enabling it, set the value back to 1 and reboot.

Step 4 - Visual Effects to Best Performance

Windows wastes GPU and CPU resources on animations and transparency effects that look nice but serve zero purpose while gaming. Strip them out:

Set-ItemProperty -Path "HKCU:\Control Panel\Desktop" -Name "DragFullWindows" -Value "0" Set-ItemProperty -Path "HKCU:\Control Panel\Desktop" -Name "MenuShowDelay" -Value "0" Set-ItemProperty -Path "HKCU:\Control Panel\Desktop\WindowMetrics" -Name "MinAnimate" -Value "0" Set-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced" -Name "TaskbarAnimations" -Value 0 Set-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\DWM" -Name "EnableAeroPeek" -Value 0

You can also hit this via System Properties → Advanced → Performance Settings → Adjust for best performance, then manually re-enable smooth edges of screen fonts if you want readable text.

Step 5 - Clear Your Temp Files

Temp files build up silently. I had nearly 4GB sitting in mine. They don't cause slowdowns by existing, but they waste drive space and can slow down antivirus scans. Clean them out:

Remove-Item $env:TEMP\* -Recurse -Force -ErrorAction SilentlyContinue

Some files will be locked by running processes and won't delete - that's fine, PowerShell skips them. Run this periodically or stick it in a scheduled task.

What I Didn't Touch

A few things worth mentioning that I left alone:

SysMain (Superfetch) - already disabled on my machine. If you're on SSDs, disable it. If you're still on HDDs, leave it on. It preloads frequently used apps into RAM and genuinely helps on spinning disks.

Windows Search indexing - leave it on. The performance impact is minimal and the search speed difference is real.

Xbox Game DVR - already off. If yours is on and you're not using it, turn it off. It hooks into every game and records in the background whether you asked it to or not.

The Result

Cleaner boot. Fewer background processes competing for resources. GPU scheduler doing its job properly. None of these are miracle fixes - but combined they add up to a machine that gets out of its own way when you're trying to play.

No overclock. No third-party tools. No sketchy registry packs downloaded from a forum. Just Windows doing what it should have done by default.
