About

The night shift for gamers

Sim racing, PC optimization, and self-hosting — documented properly, so the next person doesn't have to start from scratch.

I'm Strix. I race in sims, build and tune PCs, and run more of my own infrastructure than is strictly reasonable. STRIXEVO is where all of that lives — the guides, the build logs, the configs, and the occasional rant about a setting that was quietly eating my frames.

What STRIXEVO stands for

Three things, in order: performance (make the machine get out of your way), community (figure it out together, in the open), and optimization (no placebo tweaks — only changes that move the needle). If a post can't show the before and after, it doesn't go up.

Why a Raspberry Pi?

This whole site is served from a Raspberry Pi 3B sitting behind a Cloudflare Tunnel, running a hardened nginx. No monthly hosting bill, no cloud dashboard — just a tiny board doing real work 24/7. It's slower than it could be, and that's the point: if it runs well here, it runs well anywhere. The site is part of the build, not a mockup of one.

The stack

  • Host: Raspberry Pi 3B (1GB RAM), self-hosted at home
  • Proxy: Cloudflare Tunnel — HTTPS for free, no open ports
  • Server: nginx, hardened, static files only
  • Site: hand-written HTML/CSS/vanilla JS — no frameworks, no build step
